Persistence——Excel

这个东西DDDD，一般用来读策划同学写的Excel文档，之后有什么转Json的需求就不用跑到网页上去转换了。这里将会介绍ExcelDataReader和EPPLus两种Excel插件。

  • ExcelDataReader

    ExcelDataReader支持快速读取Excel数据，但不支持写入（基本上也不会用excel做持久化，一般只是做成数据配置表而已），读取速度和安全性相对于NPOI来说要好上不少。

    需要在Unity中添加三个依赖的dll，已上传至GitHub，放到项目中的Plugins目录下即可。PS：如果需要System.data.dll，可以到Editor目录中找到mono所对应的版本文件夹，再把里边的System.data.dll导入工程

    下面是准备好的用来展示Excel读取的数据模型

      public enum ItemType

  }

      public int id;

    • 读取Excel

      注意:

      • Excel 07 和 97-03 版本的读取方式不一样

      • 在某些.net版本下使用 GetInt32、GetFloat等（除了GetDouble、GetString）方法会抛出类型转换失效的异常，这是主要是出于性能原因，以及与 IDataRecord 接口的其他实现有关。临时的解决方案是：使用Convert.ToInt32(reader.GetValue(i))来实现类型的转换，如果常用，可以封装成拓展方法。

        //('97-2003 版本，后缀为*.xls)
//可以不用dataset，这里只是作为其中一种读取方法的展示，需要访问第几张表时用这种方法
public static List<Item> XlsReader(string exclePath)
{
  var itemlist = new List<Item>();
  using (FileStream fs = new FileStream(exclePath, FileMode.Open, FileAccess.Read))
  {
      //excel读取类
      IExcelDataReader reader = ExcelReaderFactory.CreateBinaryReader(fs);

      var result = reader.AsDataSet();//做成数据集
      var rows = result.Tables[0].Rows.Count;//获取行数，Tables[0]是第1张sheet

      for (int i = 0; i < rows; i++)
      {
          var item = new Item();

          item.id = Convert.ToInt32( result.Tables[0].Rows[i][0] );//[i][0]表示第i行第1列
          item.name = Convert.ToString(result.Tables[0].Rows[i][1]);
          item.icon = Convert.ToString(result.Tables[0].Rows[i][2]);
          item.type = (ItemType)Convert.ToInt32(result.Tables[0].Rows[i][3]);
          item.tips = Convert.ToString(result.Tables[0].Rows[i][4]);

          itemlist.Add(item);
      }
  }
  return itemlist;
}
//(2007 版本，后缀为*.xlsx)
public static List<Item> XlsxReader(string exclePath)
{
  var itemlist = new List<Item>();
  using (FileStream fs = new FileStream(exclePath, FileMode.Open, FileAccess.Read))
  {
      //excel读取类
      IExcelDataReader reader = ExcelReaderFactory.CreateOpenXmlReader(fs);

      //reader.IsFirstRowAsColumnNames(true) 第一行是否作为列名，可以用这个来跳过第一行

      //读取数据,一次读一行
      do
      {
          while (reader.Read())
          {
              var item = new Item();
              item.id = reader.GetInt32(0);//0指的是第1列
              item.name = reader.GetString(1);
              item.icon = reader.GetString(2);
              item.type = (ItemType)reader.GetInt32(3);
              item.tips = reader.GetString(4);

              itemlist.Add(item);
          }
      }while(reader.NextResult());//移动到下一张Sheet
  }
  return 
}

  • EPPlus

    EPPlus提供了更简单的Excel单元格访问方式，并且支持对Excel的读写，缺点是只支持对xlsx的读写。

    各.net版本对应的EPPlus.dll已上传至GitHub

    • 读取Excel

      注意，EPPlus中的sheet、row、cloumn的索引都是从1开始的

        public List<Item> ExcelReader(string path, int sheetIndex = 1)
  {
      var itemlist = new List<Item>();

      //也可以用Stream去获得数据
      FileInfo info = new FileInfo(path);

      using (ExcelPackage excelPackage = new ExcelPackage(info))
      {
          //sheetIndex:第几张Sheet（从1开始）
          var sheet = excelPackage.Workbook.Worksheets[sheetIndex];

          var start = sheet.Dimension.Start;//第一个单元格
          var end = sheet.Dimension.End;//最后一个单元格
          for (int row = start.Row; row <= end.Row; row++)
          {
              var item = new Item();

              //通过Cell.Text来获得每个单元格的数据
              item.id = int.Parse(sheet.Cells[row, 1].Text);
              item.name = sheet.Cells[row, 2].Text;
              item.icon = sheet.Cells[row, 3].Text;
              item.type = (ItemType)int.Parse(sheet.Cells[row, 4].Text);
              item.tips = sheet.Cells[row, 5].Text;

              itemlist.Add(item);
          }
      }
      return itemlist;
  }

    • 写入Excel

        public void ExcelWriter(string path, List<Item> itemlist, int sheetIndex = 1)
  {
      FileInfo info = new FileInfo(path);

      using (ExcelPackage excelPackage = new ExcelPackage(info))
      {
          var sheet = excelPackage.Workbook.Worksheets[sheetIndex];

          int row = 1;

          foreach(var item in itemlist)
          {
              //通过Cell.Value注入数据
              sheet.Cells[row, 1].Value = item.id;
              sheet.Cells[row, 2].Value = item.name;
              sheet.Cells[row, 3].Value = item.icon;
              sheet.Cells[row, 4].Value = (int)item.type;
              sheet.Cells[row, 5].Value = item.tips;

              row++;
          }
          //保存
          excelPackage.Save();
      }
  }
