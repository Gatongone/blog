DataSet、DataTable、DataAdapter

  • 先说一下三者之间的关系

    DataTable是DataSet的子集，用于区分一个DataSet里的不同数据，如果没有这个需求当然也可以不需要建立DataTable。

    DataAdapter就是连接数据源和DataSet之间的桥梁,如果把，数据比作水，数据源比作水源，DataSet比作水库，那么DataAdpater就相当于输水管，通过输水管将水源中的水运输到水库中进行保管；而DataTable就相当于水库中分开管理的小水库，比如这个小水库是用来输给小明家的，那个小水库是用来输给小红家的。

  • DataSet

    DataSet即数据集，是ADO.NET种的关键类，据说是为了代替DataReader而出现的，用于对数据暂存和操作，包括了查找、排序、过滤、缓存更改、处理分级数据、与XML数据交互等

    • DataTable的构造和操作

      • DataTable的构造

        先构造行，再构造列，注意行和列的声明方式有些不同

        • 插入列：

          
//方法一：先实例化再添加
//创建一个实例，构造函数为列名和列的数据类型
DataColumn column = new DataColumn("列名1",typeof(string));
//将列加入到表中
dt.Columns.Add(column);

//方法二：边实例化边添加
dt.Columns.Add("列名2",typeof(string));

        • 插入行：

          注意是没有行名的，行类似于一个栈，第一个添加到DataTable的就是第0行

          
//方法一：先实例化再添加
//创建一个实例
DataRow row = dt.NewRow();
//填写内容:
//注意0是列号不是行号
row[0]="内容";
//或者
row["列名"]="内容";
dt.Rows.Add(row);

  • DataAdapter

    DadaAdapter即适配器，一般就干两件事：从数据源获得数据（SQL中一般是通过SQL指令，在声明适配器时处理），填充数据到DataSet数据集（Fill方法）。

    • DataAdapter的声明，一般有三种方式

        //声明连接字符串
  private static string connString = @"Data Source=DESKTOP-GVLLFG5\SQLEXPRESS;Initial Catalog=DBTest;Integrated Security=True";

  //初始化连接实例
  public static SqlConnection connection = new SqlConnection(connString);

  //声明一个查询指令
  private static SqlCommand cmd = new SqlCommand("select * from DBTest", connection);

  //方法一：通过 查询指令 声明
  SqlDataAdapter da1 = new SqlDataAdapter(cmd);

  //方法二：通过 查询字符串 和 连接字符串 声明
  SqlDataAdapter da2 = new SqlDataAdapter("select * from DBTest", connString);

  //方法三：通过 查询字符串 和 连接实例 声明
  SqlDataAdapter da3 = new SqlDataAdapter("select * from DBTest", connection);

    • Fill():

      填充数据集，意思就是将数据源的数据填充到DataSet中（输水管将水源的水输到水库中）

      填充的几种方式：

        //......
  //数据库连接吧啦吧啦.....
  //......

  DataSet ds = new DataSet();
  DataTable dt = new DataTable("table_1");
  ds.Tables.Add(ds);
  SqlDataAdapter adapter = new SqlDataAdapter(cmd);

  //方式一：直接填充到数据集中
  adapter.Fill(ds);

  //方式二：直接填充到数据表中
  adater.Fill(dt);

  //方式三：填充到数据集中的某个数据表中
  adater.Fill(ds,"table_2");//如果ds中没有table_2的表则会自动创建一个名为table_2的表
